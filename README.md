# About SMPLR

Continuously generated algorithmic music coded by one human in an attempt to create without the observer effect. 

Inspired by Dadaism, Pratītyasamutpāda, Amplituhedron, cut-up and dub: Hoping that positivity constraints imposed upon combinatorial space reveals a hidden geometric structure. 

This exists, then that exists. Emergent. Enjoy!

No AIs were used nor harmed in the making of this broadcast.
